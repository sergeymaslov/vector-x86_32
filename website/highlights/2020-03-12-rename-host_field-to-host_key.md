We've renamed the [`splunk_hec` source's] `host_field` option to `host_key`.
This ensures that the `host_key` option is consistent across all sources.

## Upgrade Guide

```diff title="vector.toml"
 [sources.splunk]
   type = "splunk_hec"
-  host_field = "host"
+  host_key = "host"
```
