The `datadog` sink has been renamed to `datadog_metrics` to make way for the
upcoming `datadog_logs` sink.

## Upgrade Guide

Make the following changes in your `vector.toml` file:

```diff title="vector.toml"
 [sinks.datadog]
-  type = "datadog"
+  type = "datadog_metrics"
```

That's it!
